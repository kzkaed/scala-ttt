

##Scala Tic Tac Toe



##Setup

    $ install java (see oracle site)
    $ install scala (brew install scala or manually)
    $ install sbt (brew install sbt or manually)
    
    $ clone
    $ cd ttt
    $ sbt test (compile and run tests)

##To play:

To run Human to Human game:

    $ sbt run

##To test:

    $ sbt test
